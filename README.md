# 🐍 贪吃蛇游戏

一个现代化、功能完整的贪吃蛇游戏，使用纯HTML、CSS和JavaScript开发。

## 🎮 游戏特性

- **现代化界面**：采用渐变色彩和流畅动画设计
- **难度选择**：提供简单、中等、困难三种难度模式
- **响应式设计**：完美适配桌面和移动设备
- **触屏支持**：支持移动设备触摸控制
- **本地存储**：自动保存最高分记录
- **流畅动画**：蛇身渐变效果和食物动画
- **音效系统**：内置游戏音效（使用Web Audio API）
- **暂停功能**：随时暂停和继续游戏

## 🎯 游戏规则

1. 使用方向键或WASD控制蛇的移动方向
2. 吃到食物可以增长蛇身并获得分数
3. 避免撞到墙壁或自己的身体
4. 挑战更高难度获得更多分数

## 🕹️ 操作方式

### 键盘控制
- **方向键**：↑↓←→ 控制移动
- **WASD键**：W/A/S/D 控制移动
- **空格键**：暂停/继续游戏

### 触屏控制
- **滑动**：在游戏画布上滑动控制方向
- **按钮**：使用屏幕上的控制按钮

## 🚀 快速开始

1. 克隆仓库到本地
```bash
git clone https://github.com/yyy93yy/snake-game.git
cd snake-game
```

2. 在浏览器中打开 `index.html` 文件
或使用本地服务器：
```bash
python -m http.server 8000
```
然后访问 `http://localhost:8000`

## 🛠️ 技术栈

- **HTML5**：语义化标签和Canvas API
- **CSS3**：现代样式、动画和响应式设计
- **JavaScript ES6+**：模块化编程和现代语法
- **Web Audio API**：动态音效生成
- **LocalStorage API**：本地数据存储

## 🎨 设计特点

- **毛玻璃效果**：容器背景模糊效果
- **流畅动画**：悬停效果和过渡动画

## 🔧 自定义配置

游戏支持以下自定义选项：

- **游戏速度**：通过难度选择器调整
- **画布大小**：在CSS中修改canvas尺寸
- **颜色主题**：在CSS中修改渐变色彩
- **音效音量**：在JavaScript中调整音量参数

## 📈 分数系统

- **简单模式**：每个食物10分
- **中等模式**：每个食物20分
- **困难模式**：每个食物30分

最高分自动保存在浏览器本地存储中。
