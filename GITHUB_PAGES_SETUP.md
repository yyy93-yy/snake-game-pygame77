# 🚀 GitHub Pages 启用步骤详细指南

## 📋 第一步：访问GitHub仓库

1. **打开浏览器，访问你的GitHub仓库**：
   ```
   https://github.com/yyy93-yy/snake-game-pygame77
   ```

## ⚙️ 第二步：进入设置页面

2. **点击仓库导航栏中的 "Settings" 按钮**
   - 位置：仓库页面右上角，仓库名称下方
   - 图标：⚙️ 齿轮图标

3. **在左侧菜单中找到 "Pages" 选项**
   - 滚动左侧菜单，找到 "Code and automation" 部分
   - 点击 "Pages"

## 🔧 第三步：配置GitHub Pages设置

4. **在 "Source" 部分进行以下设置**：

   **a) 选择部署源**
   - 点击 "Source" 下拉菜单
   - 选择：**"Deploy from a branch"**

   **b) 选择分支**
   - 点击 "Branch" 下拉菜单  
   - 选择：**"main"**

   **c) 选择文件夹**
   - 在 "Folder" 选项中
   - 选择：**"/ (root)"**

5. **点击 "Save" 按钮保存设置**
   - 按钮位置：页面右下角
   - 颜色：绿色

## ⏱️ 第四步：等待构建完成

6. **查看构建状态**
   - 保存后，页面会显示构建状态
   - 可能显示：**"Your site is being built"**
   - 等待1-10分钟完成构建

7. **构建成功后会显示**
   - 绿色提示：**"Your site is published at..."**
   - 显示你的网站地址

## 🌍 第五步：访问你的游戏

8. **访问游戏地址**
   - 主要地址：https://yyy93-yy.github.io/snake-game-pygame77/
   - 测试页面：https://yyy93-yy.github.io/snake-game-pygame77/test.html

## 📱 第六步：验证和分享

9. **测试游戏功能**
   - 检查游戏是否能正常加载
   - 测试键盘控制（方向键或WASD）
   - 测试开始、暂停、重新开始功能

10. **分享给朋友**
    - 复制网站链接
    - 可以在任何设备上访问

## 🔍 常见问题和解决方案

### 问题1：找不到Pages选项
- **解决**：确保仓库是公开的（Public）
- **检查**：Settings → General → Danger Zone → Change repository visibility

### 问题2：构建失败
- **解决**：检查Actions页面查看详细错误
- **地址**：https://github.com/yyy93-yy/snake-game-pygame77/actions

### 问题3：仍然404错误
- **解决**：等待更长时间（最多30分钟）
- **检查**：清除浏览器缓存，尝试无痕浏览

### 问题4：分支选择中没有main选项
- **解决**：确保main分支存在且有文件
- **检查**：https://github.com/yyy93-yy/snake-game-pygame77/branches
